#### 介绍

这是一个基于Layui+SSM的学生信息管理系统，很经典的系统。其中学生管理和成绩管理没开发。

#### 技术体系

前端：Layui，HTML，CSS，jQuery等。

后端：Spring，SpringMVC，Mybatis

数据库：MySQL
